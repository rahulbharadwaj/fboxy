fBoxy beta

fBoxy is a jQuery plugin which adds the ability to load content into a lightbox style popup box. The content can be loaded from another element on page, or any other url (via AJAX or iFrame).

Usage

  1. Download and include jQuery v1.7+
  2. Download and include following files (Download All):
    • plugin/script.js
    • plugin/style.css
    • plugin/ring.gif
  3. User $(selector).fboxy();
This content has been loaded from a <div> with the id "sample1". The original div is hidden using CSS.

Options

  • showClose (default: true)
    Display a close button above the boxy window.
  • showTitle (default: true)
    Uses the "title" attribute of the initiating element to display a title above the boxy window.
  • closeEsc (default: true)
    The boxy window would close if esc key is pressed.
  • closeBg (default: true)
    The boxy window would close if clicked outside the boxy window.
  • iframe (default: false)
    If case of url, the link would be loaded inside an iframe.
  • zindex (default: 1000)
    The z-index of the boxy window.
